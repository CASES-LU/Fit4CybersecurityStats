Fit4Cybersecurity

Fit4Cybersecurity is a self-assessment tool by CASES to help business owners implement a better cybersecurity strategy.
Its main goal is to allow users to quickly assess their information security maturity in different areas that CASES considers critical.

As an example this tool allows instantiation of the following self-assessment websites:

  • Fit4Cybersecurity,
  • Fit4Privacy,
  • OperatorSurvey.

You can visit this page which is federating statistical data from the previously mentioned Fit4Cybersecurity instances.
